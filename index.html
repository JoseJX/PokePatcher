The PokePatcher

I'm really getting tired of patching Pokemon ROM variants to the .Pocket format. Luckily, they're not too complicated to patch, so the process can be automated! I Hope!

This javascript program will run locally on your PC. The file you create is local to your PC and will not be sent anywhere over the internet.

It will patch any Pokemon Variant for use on the Analogue Pocket. If there are bugs, or if you have any issues, please report them on the github page for this code.

Select your ROM file using the control below and then save the converted .pocket file to /GB Studio/. This tool will trim the filename to less than 32 characters to make sure there aren't any issues with saving, please make sure to keep the filename short.
